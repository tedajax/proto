Joystick:isGamepadDown (한국어)

Available since LÖVE 0.9.0
This function is not supported in earlier versions.

가상 게임패드 호환 조이스틱 버튼이 눌렸는지 확인합니다. 조이스틱이 가상 게임패드로 여겨지지 않거나 연결되지 않았을 경우, 이 함수는 항상 0을 리턴합니다.

함수

형식

anyDown = Joystick:isGamepadDown( button1, button2, button3, ... )

매개변수

GamepadButton (한국어) buttonN
확인할 가상 게임패드 호환 조이스틱 버튼.

리턴값

boolean (한국어) anyDown
단 하나의 버튼이라도 눌려있으면 true.
